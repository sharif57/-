1. `npm install` চালান
2. `npm run dev` দিয়ে শুরু করুন
3. http://localhost:3000 খুলুন
4. সব বৈশিষ্ট্য পরীক্ষা করুন
5. `vercel deploy` দিয়ে ডিপ্লয় করুন
