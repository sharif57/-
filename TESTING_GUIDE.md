# পূর্ণ পরীক্ষা এবং যাচাইকরণ গাইড

## সব বৈশিষ্ট্য পরীক্ষা করার ধাপসমূহ

### 1. হোম পেজ পরীক্ষা
```
✓ সব ১১৪ সূরা দেখা যাচ্ছে
✓ সার্চ বক্স কাজ করছে
✓ সার্চ ফলাফল তাৎক্ষণিক আসছে
✓ বুকমার্ক বোতাম কাজ করছে
✓ কার্ড হোভার অ্যানিমেশন কাজ করছে
✓ থিম টগেল কাজ করছে
```

### 2. সূরা বিস্তারিত পৃষ্ঠা পরীক্ষা
```
✓ কার্ড ক্লিক করে বিস্তারিত পৃষ্ঠা খোলা যাচ্ছে
✓ আরবি পাঠ দেখা যাচ্ছে
✓ বাংলা অনুবাদ দেখা যাচ্ছে
✓ সূরা নম্বর এবং আয়াত নম্বর দেখা যাচ্ছে
✓ ফিরে যান বোতাম কাজ করছে
✓ বুকমার্ক টগল কাজ করছে
```

### 3. অডিও প্লেয়ার পরীক্ষা
```
✓ Play বোতাম কাজ করছে - অডিও চালু হচ্ছে
✓ Pause বোতাম কাজ করছে - অডিও থেমে যাচ্ছে
✓ Progress bar কাজ করছে - টানা যাচ্ছে
✓ সময় প্রদর্শিত হচ্ছে (বর্তমান / মোট)
✓ ভলিউম স্লাইডার কাজ করছে
✓ ভলিউম প্রতিশত দেখা যাচ্ছে
✓ অডিও শেষ হলে স্বয়ংক্রিয় থামছে
```

### 4. অনুবাদ বৈশিষ্ট্য পরীক্ষা
```
✓ অনুবাদ দেখান/লুকান বোতাম কাজ করছে
✓ অনুবাদ সঠিকভাবে প্রদর্শিত হচ্ছে
✓ অনুবাদ হাইলাইট করা আছে
✓ প্রতিটি আয়াতের অনুবাদ আছে
```

### 5. কপি বৈশিষ্ট্য পরীক্ষা
```
✓ কপি বোতাম প্রতিটি আয়াতে আছে
✓ ক্লিক করলে টেক্সট ক্লিপবোর্ডে কপি হয়
✓ সফল কপির বার্তা দেখা যায়
✓ কপি আইকন চেঞ্জ হয় ✓ তে
```

### 6. বুকমার্ক সিস্টেম পরীক্ষা
```
✓ হোম পেজে সূরা বুকমার্ক করা যায়
✓ বুকমার্ক সংরক্ষিত হয়
✓ পৃষ্ঠা রিফ্রেশ করলেও বুকমার্ক থাকে
✓ বুকমার্ক আনবুকমার্ক করা যায়
✓ বিস্তারিত পৃষ্ঠায় বুকমার্ক স্ট্যাটাস সিঙ্ক থাকে
```

### 7. সার্চ ফাংশনালিটি পরীক্ষা
```
✓ বাংলা নামে সার্চ করা যায়
✓ ইংরেজি নামে সার্চ করা যায়
✓ নম্বরে সার্চ করা যায়
✓ আংশিক নামে খোঁজা যায়
✓ কোনো ফলাফল না পেলে বার্তা দেখা যায়
```

### 8. ডিজাইন এবং ইউজার অভিজ্ঞতা পরীক্ষা
```
✓ সব পৃষ্ঠা প্রতিক্রিয়াশীল দেখাচ্ছে
✓ মোবাইলে সঠিকভাবে প্রদর্শিত হচ্ছে
✓ ট্যাবলেটে সঠিকভাবে প্রদর্শিত হচ্ছে
✓ ডেস্কটপে সঠিকভাবে প্রদর্শিত হচ্ছে
✓ রঙ স্কিম সামঞ্জস্যপূর্ণ
✓ ফন্ট সঠিকভাবে দেখা যাচ্ছে
✓ অ্যানিমেশন মসৃণ
```

### 9. পারফরম্যান্স পরীক্ষা
```
✓ প্রথম লোড দ্রুত (< 3 সেকেন্ড)
✓ সূরা তালিকা দ্রুত লোড হয়
✓ বিস্তারিত পৃষ্ঠা দ্রুত খোলে
✓ সার্চ তাৎক্ষণিক ফলাফল দেয়
✓ অডিও মসৃণভাবে চলে
```

### 10. ব্রাউজার সামঞ্জস্যতা পরীক্ষা
```
✓ Chrome-এ কাজ করছে
✓ Firefox-এ কাজ করছে
✓ Safari-তে কাজ করছে
✓ Edge-এ কাজ করছে
✓ মোবাইল ব্রাউজারে কাজ করছে
```

## ত্রুটি নিষ্পত্তি

### অডিও না চলালে
1. ব্রাউজার কনসোল খুলুন (F12)
2. "[v0] Audio play error:" খুঁজুন
3. নেটওয়ার্ক ট্যাব চেক করুন - অডিও ফাইল লোড হচ্ছে?
4. CORS ত্রুটি থাকলে - সার্ভার কনফিগ চেক করুন

### সার্চ না হলে
1. সার্চ ইনপুট মূল্য চেক করুন
2. কনসোলে ত্রুটি খুঁজুন
3. ফিল্টারিং লজিক যাচাই করুন

### বুকমার্ক না সংরক্ষিত হলে
1. localStorage কি সক্ষম? (Chrome DevTools > Application > LocalStorage)
2. ডেটা "quran_bookmarks" কী-তে সংরক্ষিত আছে?

### থিম না পরিবর্তন হলে
1. Header.tsx-এ থিম পরিবর্তন কোড চেক করুন
2. ডকুমেন্ট এলিমেন্টে "dark" ক্লাস যোগ হচ্ছে?
